
# Политика конфиденциальности — Exotic Assistant Pro

**Обновлено:** 2025-12-19

## Какие данные обрабатываются
- Расширение хранит локально настройки (задержки, тексты поиска, включение уведомлений), статистику кликов и лицензию.
- Для работы лицензии и Telegram-интеграции расширение отправляет ID пользователя и технические параметры на сервер лицензий и Telegram (только для активации и уведомлений).
- Данные не передаются третьим лицам и не используются для рекламы.

## Разрешения и их обоснование
- `storage`: хранение пользовательских настроек, статистики и лицензии локально.
- `activeTab` / `scripting`: автоматизация действий на целевых страницах.
- `notifications`: локальные уведомления о кликах, статусе лицензии и действиях.
- `alarms`: ежедневный сброс статистики и периодические проверки лицензии.
- `host_permissions`: доступ только к страницам exotic.company и поддоменам для автоматизации.
- `remote code`: не используется для загрузки стороннего кода, только для связи с сервером лицензий и Telegram.

## Передача данных
- Персональные, финансовые, медицинские, геолокационные данные и история действий пользователя не собираются.
- Данные не продаются и не используются вне работы основного функционала.

## Контроль пользователя
- Отключить расширение или очистить его данные можно через менеджер расширений браузера.
- Telegram-интеграция отключается в настройках расширения.

## Контакты
- Вопросы по политике: через страницу расширения в Chrome Web Store или через Telegram-бот поддержки.
