## 🚀 Возможности

- **🖱️ Автоматизация кликов** — автоматически нажимает на кнопки "В работу" при появлении заявок
- **📱 Telegram уведомления** — мгновенные уведомления о новых заявках прямо в Telegram
- **📊 Статистика** — подробная статистика по кликам: за день, неделю, месяц
- **⚙️ Настраиваемость** — гибкие настройки интервалов, задержек и целевых кнопок
- **💱 Калькулятор курса** — автоматический расчёт курса USDT на странице заявки
- **🔒 Безопасность** — все данные хранятся локально, минимальная передача данных

## 📥 Установка

### Chrome Web Store (рекомендуется)

1. Перейдите в [Chrome Web Store](https://chrome.google.com/webstore)
2. Найдите "Exotic Operator Assistant"
3. Нажмите "Добавить в Chrome"

### Ручная установка (для разработчиков)

1. Скачайте или клонируйте репозиторий
2. Откройте `chrome://extensions/`
3. Включите "Режим разработчика"
4. Нажмите "Загрузить распакованное расширение"
5. Выберите папку с расширением

## 📱 Telegram уведомления

1. Откройте настройки расширения
2. Нажмите "Подключить Telegram"
3. Перейдите по ссылке к боту
4. Получите **12 часов** бесплатного пробного периода!

## 🤖 Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Запуск бота и главное меню |
| `/status` | Статус подписки |
| `/devices` | Управление устройствами |
| `/groups` | Управление группами (L2, L3) |
| `/ref` | Реферальная программа |

## 🎁 Реферальная программа

- Новый пользователь получает **24 часа** пробного периода
- Вы получаете **+1 день** когда реферал оформит подписку

## 🔒 Конфиденциальность

- Расширение **НЕ** собирает личные данные
- Статистика хранится только на вашем устройстве
- Telegram используется только для уведомлений

Подробнее: [Политика конфиденциальности](PRIVACY_POLICY.md)
