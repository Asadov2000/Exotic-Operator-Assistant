# Политика конфиденциальности - Exotic Operator Assistant

**Дата вступления в силу:** Январь 2025

## Введение

Расширение "Exotic Operator Assistant" (далее - "Расширение") уважает вашу конфиденциальность. Эта политика описывает, какие данные собираются, как они используются и защищаются.

## Какие данные собираются

### Локальные данные (хранятся только на вашем устройстве):
- Настройки расширения
- Статистика кликов и активности
- История уведомлений

### Данные, передаваемые на сервер:
- Уникальный идентификатор установки (deviceId) - случайно сгенерированный UUID
- Telegram Chat ID (при подключении уведомлений)
- Telegram Username (при подключении уведомлений)

## Как используются данные

- **Локальные данные** используются исключительно для работы расширения и никуда не передаются
- **DeviceId** используется для идентификации устройства в рамках лицензии
- **Telegram данные** используются для отправки вам уведомлений о заявках

## Какие данные НЕ собираются

- Личная информация (имя, email, телефон)
- Финансовые данные
- Пароли и учётные данные
- Содержимое страниц, которые вы просматриваете
- История браузера

## Хранение данных

- Локальные данные хранятся в хранилище браузера Chrome (chrome.storage.local)
- Серверные данные хранятся на защищённых серверах Cloudflare Workers

## Передача данных третьим лицам

Мы **НЕ** передаём, не продаём и не раскрываем ваши данные третьим лицам, за исключением:
- Telegram Bot API (для отправки уведомлений по вашему запросу)

## Удаление данных

- Локальные данные удаляются при удалении расширения
- Для удаления данных с сервера отправьте команду /disconnect боту в Telegram

## Безопасность

Мы используем следующие меры безопасности:
- HTTPS для всех соединений
- Cloudflare защита от DDoS и атак
- Минимальный сбор данных

## Изменения в политике

Мы можем обновлять эту политику. Об изменениях будет сообщено через обновление расширения.
